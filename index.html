X65

Modern 8-bit microcomputer

READY
█

X65 is an effort to build an 8-bit microcomputer for the modern era. It uses best of breed components and strives to keep 8-bit feeling, while being usable for daily basis computing activities.

Components

  • CPU: WDC 65C816, 10MHz
  • RAM: 32MB (SDR SDRAM)
  • Video: Atari CGIA with CBM TED video modes, DVI-I
  • Sound: Yamaha OPL3-L YMF289, Stereo RCA, S/PDIF
  • Input/Output:
    • WDC 65C22 VIA
    • SPI
  • Ports:
    • SD Card slot
    • 2x DE-9 Joystick port (SEGA wiring)
    • 4x USB port

Hardware is to be based on MIST design providing implementation of non-existing chips and glue to modern digital interfaces (USB, DVI, S/PDIF).

Why?

Because we need "Understandable Computers" to make computers unstandable.
